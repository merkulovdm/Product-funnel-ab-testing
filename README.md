# Product Funnel & A/B Testing Analysis

## Overview

This project analyzes user behavior within a product funnel and evaluates the impact of UI changes through A/B testing.

The goal was to identify major drop-off points in the funnel and determine whether a new font affected user conversion.

## Dataset

The dataset contains:
- user event logs
- product funnel events
- experiment group assignments

Key events:
- app launch
- product screen view
- cart interaction
- successful payment

## Analysis Scope

The analysis focused on:
- conversion funnel analysis
- user drop-off points
- A/A/B testing
- statistical significance validation

# Key Findings

- Only ~48% of users reached successful payment
- The largest drop-off occurred between the main screen and the offers screen
- Users who added products to the cart converted successfully in ~95% of cases
- The checkout process itself showed strong conversion performance

The analysis indicated that the main issue was located in the upper part of the funnel rather than in the payment flow.

# A/B Test Results

No statistically significant differences were found between control and experimental groups.
The font change did not negatively impact user behavior or conversion metrics.

# Recommendations

- Improve the main screen value proposition
- Optimize navigation toward the offers section
- Continue testing onboarding and UI changes
- Focus future experiments on upper-funnel conversion improvements
